# Music Player with Playlist

## Project Description
This project is a Music Player with Playlist developed using Java, JDBC, and MySQL Database.

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Features
- Add Songs
- View Songs
- Create Playlist
- Play Songs
- Delete Songs

## Database
Import the musicplayer.sql file into MySQL Workbench.
